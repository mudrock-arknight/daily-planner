# Product

## Register

product

## Users

个人大学生用户，日常使用规划学习和生活进度，在手机和桌面端都需要使用。用户每天打开多次，进行打卡、查看计划、更新进度。

## Product Purpose

个人计划管理工具，核心功能包括：
- 周计划安排（9大长期目标）
- 每日时间块规划与自动完成
- 学习类型任务手动打卡
- 学习时长统计与进度展示
- 课程表集成
- 六级考试倒计时

帮助用户保持学习节奏，跟踪长期目标进度，自动化非学习任务完成标记。

## Brand Personality

活力现代，清晰简洁，专注高效。

3 words: **Modern · Focused · Clean**

## Anti-references

- 不要 AI 紫色渐变背景
- 不要花哨动画
- 不要大量玻璃拟态效果
- 不要过度装饰，功能优先
- 不要不必要的视觉干扰

## Design Principles

1. **功能优先**：设计服务于任务，工具应该消失在任务背后
2. **清晰层次**：通过对比度和间距建立明确视觉层次，重要信息一眼可见
3. **适度活力**：保持现代感但不花哨，少量动作用于状态反馈
4. **一致体验**：所有页面保持统一的组件语言和交互模式

## Accessibility & Inclusion

- WCAG AA 对比度要求
- 尊重系统减少 motion 设置
- 移动端优先，适配手机屏幕

# Dial Settings (for design)

```json
{
  "DESIGN_VARIANCE": 5,
  "MOTION_INTENSITY": 3,
  "VISUAL_DENSITY": 5
}
```

These dials guide all design decisions:
- **Variance 5**: Predictable layout, mostly symmetrical but with subtle variation
- **Motion 3**: Static with minimal transitions, only state changes animate
- **Density 5**: Standard spacing for a daily app, not too crowded not too airy
