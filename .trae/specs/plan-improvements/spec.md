# 个人规划助手改进 - Product Requirement Document

## Overview
- **Summary**: 优化现有个人规划网站，使其更加实用、合理，提供长期大学规划功能，修复数据统计和计划逻辑问题
- **Purpose**: 解决当前计划不合理、统计不清晰、缺少长期规划等问题，打造一个真正能帮助用户管理大学生活的工具
- **Target Users**: 大学生，特别是需要平衡考试准备和长期规划的学生

## Goals
- **修复数据统计功能** - 让统计数据有意义、清晰易懂
- **实现长期大学规划** - 不只是六级，而是涵盖用户大学期间的全面成长目标
- **优化时间块标记逻辑** - 修复"已完成"判断逻辑错误
- **合理安排日常时间** - 考虑吃饭、洗漱等生活时间
- **英语学习时长合理** - 每天约2小时，直到六级考试

## Non-Goals (Out of Scope)
- 不要过于复杂的功能，保持简单实用
- 不添加需要大量数据输入的功能
- 不添加用户身份验证和多用户功能（目前仅个人使用）

## Background & Context
现有网站已基本功能完成，但存在以下问题：
1. 数据统计显示内容不明确
2. 周计划主要围绕六级备考，缺少长期规划
3. 时间块标记"已完成"逻辑有问题（未来时间也显示已完成）
4. 学习时间安排不现实，没有考虑日常活动

用户已在日记中明确列出大学期间的长期成长目标：
1. **英语** - 刷高分、考六级、准备雅思
2. **人工智能** - 认为这是所有行业的根基
3. **投资知识** - 股票、量化投资、基金
4. **编程能力** - 提升开发技能
5. **单片机** - 建模设计→3D打印→单片机编程链路
6. **打字速度** - 很重要
7. **练字** - 考研也需要
8. **艺术爱好** - 乐器（钢琴/吉他）或唱歌
9. **普通话** - 从三甲提升到二乙

## Functional Requirements
- **FR-1**: 修复"已完成"时间块的判断逻辑，只对已过去的时间块显示已完成
- **FR-2**: 优化数据统计页面，显示有意义的统计信息
- **FR-3**: 更新周计划生成逻辑，包含日常活动（吃饭、洗漱等）
- **FR-4**: 实现长期大学目标管理功能，将用户列出的9大长期目标纳入规划
- **FR-5**: 合理安排英语学习时间，每天约2小时

## Non-Functional Requirements
- **NFR-1**: 所有功能保持响应式，适配移动端
- **NFR-2**: 数据操作快速响应，用户体验流畅
- **NFR-3**: 界面设计保持一致的现代风格

## Constraints
- **Technical**: 使用现有技术栈（React + TypeScript + Tailwind + Supabase）
- **Business**: 保持免费部署，使用现有Supabase和GitHub Pages
- **Dependencies**: 依赖现有数据库结构和API

## Assumptions
- 用户每天有固定的作息时间
- 用户需要平衡课程、学习、生活等多方面
- 长期目标需要分阶段实现

## Acceptance Criteria

### AC-1: 修复"已完成"时间块逻辑
- **Given**: 用户打开计划页面
- **When**: 查看不同日期的时间块
- **Then**: 只有当前时间已过的时间块才显示"已完成"，未来时间块不显示
- **Verification**: `programmatic`
- **Notes**: 修复时间比较逻辑，使用正确的日期+时间组合

### AC-2: 优化数据统计页面
- **Given**: 用户访问统计页面
- **When**: 查看统计数据
- **Then**: 显示清晰的学习时长统计、打卡完成率、目标进度等
- **Verification**: `human-judgment`
- **Notes**: 统计数据要直观易懂，有图表或进度条展示

### AC-3: 合理安排日常时间
- **Given**: AI生成或用户查看周计划
- **When**: 看时间安排
- **Then**: 包含吃饭、洗漱等日常活动，学习时间合理（每天英语约2小时）
- **Verification**: `human-judgment`

### AC-4: 长期大学规划
- **Given**: 用户查看周计划
- **When**: 看目标和重点
- **Then**: 包含用户列出的9大长期目标（英语、人工智能、投资、编程、单片机、打字、练字、艺术爱好、普通话），不只是六级考试
- **Verification**: `human-judgment`
- **Notes**: 目标分类展示，可按优先级和时间跨度筛选

### AC-5: 英语学习时间合理
- **Given**: 查看每日安排
- **When**: 看英语学习时间
- **Then**: 每天英语学习时间约2小时，合理分配
- **Verification**: `human-judgment`
