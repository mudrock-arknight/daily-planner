# 🚀 部署到公网指南

## 方案一：使用 Vercel 部署（推荐，最简单）

### 第一步：注册 GitHub 账号
1. 访问 https://github.com
2. 注册一个免费账号（如果没有）

### 第二步：创建 GitHub 仓库
1. 登录 GitHub
2. 点击右上角的 "+" → "New repository"
3. 仓库名称：`daily-planner`（或其他你喜欢的名字）
4. 选择 "Public" 或 "Private"（都可以）
5. **不要**勾选 "Initialize this repository"
6. 点击 "Create repository"

### 第三步：上传代码到 GitHub
在项目目录下，打开终端（PowerShell），依次执行：

```bash
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/你的用户名/你的仓库名.git
git push -u origin main
```

注意：把上面的“你的用户名”和“你的仓库名”换成真实的！

### 第四步：使用 Vercel 部署
1. 访问 https://vercel.com
2. 用 GitHub 账号登录
3. 点击 "New Project"
4. 选择你刚创建的 `daily-planner` 仓库
5. 点击 "Import"
6. 保持默认设置，点击 "Deploy"
7. 等待1-2分钟，部署完成！

### 第五步：完成！
部署成功后，你会得到一个类似 `https://daily-planner-xxx.vercel.app` 的网址，手机和电脑都可以随时访问！

---

## 方案二：使用 Netlify 部署（备选）
如果 Vercel 不行，可以用 Netlify：
1. 访问 https://netlify.com
2. 连接 GitHub 账号
3. 导入你的仓库
4. 点击部署，同样简单！

---

## 后续更新代码
修改代码后，只需：
```bash
git add .
git commit -m "更新描述"
git push
```
Vercel 会自动重新部署！
